# Agent Instructions

Use Beads (bd) as the default task and plan system for this repo.
- If starting work, run `bd ready` to pick the next unblocked task.
- When creating work, capture tasks with `bd create` and link dependencies with `bd dep add`.
- Keep tasks updated: add notes, set status, and close when done.
- If no tasks exist, run `bd quickstart` and create an initial task graph.
