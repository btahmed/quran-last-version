## 2024-05-18 - Tooltip Pattern
**Learning:** Adding `title` attributes to icon-only buttons improves accessibility for mouse users, and `aria-label` improves it for screen reader users. The application frequently uses native HTML `<button>` elements with emojis for icon-only controls.
**Action:** When modifying or adding these components, always append appropriate native `aria-label` and `title` attributes localized in Arabic.
