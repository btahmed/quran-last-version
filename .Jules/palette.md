## 2024-05-09 - Accessibility of Emoji-only Buttons in Arabic Application
**Learning:** This application heavily relies on emoji-only buttons for actions like delete (🗑️) and close (✕), particularly in administrative panels. These lack context for screen readers and can be difficult for keyboard-only users to understand without visual cues. Since the app is entirely in Arabic, standard automated a11y recommendations for English ARIA labels apply, but must be culturally and linguistically localized.
**Action:** When working on this UI, proactively scan for icon/emoji-only `<button>` tags without text content. Always add `aria-label` and `title` attributes that are strictly localized in Arabic (e.g., `aria-label="إغلاق"` instead of `aria-label="Close"`). This ensures a seamless and native experience for Arabic screen reader users.
